Epsilon-Delta Definition of Limits: Study Guide

Introduction

The epsilon-delta (ε-δ) definition of a limit is the rigorous, formal way to define what we mean when we say "the limit of f(x) as x approaches a equals L." While you may have learned limits through graphs and intuition, this definition provides the mathematical foundation that makes calculus precise.

The Formal Definition

Definition: We say that the limit of f(x) as x approaches a equals L, written as:

lim[x→a] f(x) = L

if and only if:

For every ε > 0, there exists a δ > 0 such that whenever 0 < |x - a| < δ, we have |f(x) - L| < ε.

Breaking Down the Definition

Let's understand each part:

  • ε (epsilon): A small positive number representing how close f(x) must be to L
  • δ (delta): A small positive number representing how close x must be to a
  • |f(x) - L| < ε: The distance between f(x) and L is less than ε
  • 0 < |x - a| < δ: The distance between x and a is less than δ, but x ≠ a

The Logic

The definition says: "No matter how small a tolerance ε you give me around L, I can find a corresponding tolerance δ around a such that all function values within δ of a (except at a itself) will be within ε of L."

Geometric Interpretation

Imagine drawing horizontal lines at y = L + ε and y = L - ε, creating a "tolerance band" around L. The definition says we can always find vertical lines at x = a - δ and x = a + δ such that the graph of f(x) stays within the tolerance band for all x-values between these vertical lines (except at x = a).

Example 1: Simple Linear Function

Prove: lim[x→3] (2x + 1) = 7

Solution:

We need to show that for any ε > 0, we can find δ > 0 such that:

If 0 < |x - 3| < δ, then |(2x + 1) - 7| < ε

First, let's simplify the expression we want to be small:

|(2x + 1) - 7| = |2x - 6| = 2|x - 3|

We want: 2|x - 3| < ε

This means: |x - 3| < ε/2

So if we choose δ = ε/2, then whenever 0 < |x - 3| < δ, we have:

|(2x + 1) - 7| = 2|x - 3| < 2(ε/2) = ε ✓

Answer: For any ε > 0, choose δ = ε/2.

Example 2: Quadratic Function

Prove: lim[x→2] x² = 4

Solution:

We need: |x² - 4| < ε whenever 0 < |x - 2| < δ

Factor the expression: |x² - 4| = |x - 2||x + 2|

The challenge is that |x + 2| isn't constant. We need to bound it.

If we restrict x to be within 1 unit of 2 (i.e., 1 < x < 3), then:

3 < x + 2 < 5, so |x + 2| < 5

Now: |x² - 4| = |x - 2||x + 2| < 5|x - 2|

For this to be less than ε, we need: 5|x - 2| < ε, so |x - 2| < ε/5

Answer: Choose δ = min(1, ε/5). This ensures both that |x + 2| < 5 and that |x² - 4| < ε.

Example 3: Rational Function

Prove: lim[x→1] (3x + 2)/(x + 1) = 5/2

Solution:

We need: |(3x + 2)/(x + 1) - 5/2| < ε whenever 0 < |x - 1| < δ

Simplify the expression:

|(3x + 2)/(x + 1) - 5/2| = |(2(3x + 2) - 5(x + 1))/(2(x + 1))| = |(6x + 4 - 5x - 5)/(2(x + 1))| = |x - 1|/(2|x + 1|)

We need to bound |x + 1| from below. If we keep x within 1/2 of 1 (i.e., 1/2 < x < 3/2), then:

3/2 < x + 1 < 5/2, so |x + 1| > 3/2

Therefore: |x - 1|/(2|x + 1|) < |x - 1|/(2 · 3/2) = |x - 1|/3

For this to be less than ε: |x - 1|/3 < ε, so |x - 1| < 3ε

Answer: Choose δ = min(1/2, 3ε).

Example 4: Function with a Jump (Limit Does Not Exist)

Show: lim[x→0] |x|/x does not exist

Analysis:

f(x) = |x|/x = {1 if x > 0, -1 if x < 0}

For the limit to exist and equal some value L, we would need to find δ > 0 such that for any ε > 0:

If 0 < |x| < δ, then ||x|/x - L| < ε

But for any δ > 0, we can find x₁ and x₂ with 0 < |x₁|, |x₂| < δ such that:

  • f(x₁) = 1 (when x₁ > 0)
  • f(x₂) = -1 (when x₂ < 0)

This means |f(x₁) - f(x₂)| = |1 - (-1)| = 2

No single value L can be within ε of both 1 and -1 when ε < 1.

Example 5: More Complex Function

Prove: lim[x→0] x²sin(1/x) = 0 (when x ≠ 0)

Solution:

Since sin(1/x) is bounded: -1 ≤ sin(1/x) ≤ 1 for x ≠ 0

Therefore: |x²sin(1/x) - 0| = |x²||sin(1/x)| ≤ |x²| · 1 = x²

We want x² < ε, which gives us |x| < √ε

Answer: For any ε > 0, choose δ = √ε. Then whenever 0 < |x| < δ, we have |x²sin(1/x)| < ε.

Common Strategies

  1. Work backwards: Start with |f(x) - L| and manipulate it to find a relationship with |x - a|
  2. Factor and bound: For polynomials and rational functions, factor expressions and bound terms that aren't directly related to |x - a|
  3. Use restrictions: Sometimes restrict the domain (choose δ ≤ some constant) to make bounding easier
  4. Triangle inequality: |a + b| ≤ |a| + |b| and ||a| - |b|| ≤ |a - b| can be helpful

Practice Problems

  1. Prove: lim[x→5] (3x - 7) = 8
  2. Prove: lim[x→1] x³ = 1
  3. Prove: lim[x→2] 1/(x + 1) = 1/3
  4. Show that: lim[x→0] x sin(1/x²) = 0 (when x ≠ 0)

Tips for Success

  • Start simple: Master linear functions before moving to polynomials and rational functions
  • Show your work: Always show how you choose δ in terms of ε
  • Check your answer: Verify that your choice of δ actually works
  • Practice regularly: The more problems you solve, the more patterns you'll recognize
  • Understand the geometry: Visualizing the ε-δ process helps build intuition

Common Mistakes to Avoid

  • Forgetting the restriction 0 < |x - a| (x cannot equal a)
  • Not properly bounding expressions when dealing with products
  • Choosing δ too large and not ensuring all necessary conditions are met
  • Confusing the roles of ε and δ

Remember: The epsilon-delta definition is the foundation of mathematical analysis. While it may seem abstract at first, mastering it will deepen your understanding of continuity, derivatives, and the entire structure of calculus.
