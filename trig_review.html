Trigonometry Review for Calculus

Master these essential concepts before diving into calculus

🔺 Right Triangle Trigonometry

Primary Functions: sin θ = opposite/hypotenuse = b/c
cos θ = adjacent/hypotenuse = a/c
tan θ = opposite/adjacent = b/a

Reciprocal Functions: csc θ = 1/sin θ = c/b
sec θ = 1/cos θ = c/a
cot θ = 1/tan θ = a/b

🔗 Essential Trigonometric Identities

Pythagorean Identities: sin²θ + cos²θ = 1
1 + tan²θ = sec²θ
1 + cot²θ = csc²θ
Even/Odd Functions: sin(-θ) = -sin θ (odd function)
cos(-θ) = cos θ (even function)
tan(-θ) = -tan θ (odd function)
Sum and Difference Formulas: sin(A ± B) = sin A cos B ± cos A sin B
cos(A ± B) = cos A cos B ∓ sin A sin B
tan(A ± B) = (tan A ± tan B)/(1 ∓ tan A tan B)
Double Angle Formulas: sin(2θ) = 2 sin θ cos θ
cos(2θ) = cos²θ - sin²θ
cos(2θ) = 2cos²θ - 1 = 1 - 2sin²θ
tan(2θ) = 2 tan θ/(1 - tan²θ)

↩️ Inverse Trigonometric Functions

Domains and Ranges: arcsin x: Domain [-1,1], Range [-π/2, π/2]
arccos x: Domain [-1,1], Range [0, π]
arctan x: Domain ℝ, Range (-π/2, π/2)

Key Properties: sin(arcsin x) = x for x ∈ [-1,1]
arcsin(sin x) = x for x ∈ [-π/2, π/2]
cos(arccos x) = x for x ∈ [-1,1]
arccos(cos x) = x for x ∈ [0, π]
tan(arctan x) = x for all real x
arctan(tan x) = x for x ∈ (-π/2, π/2)

Key Function Properties:

Regular Trigonometric Functions:
• sin x and cos x: Period = 2π, Range = [-1,1], Continuous everywhere
• tan x: Period = π, Range = ℝ, Vertical asymptotes at x = π/2 + nπ
Inverse Trigonometric Functions:
• arcsin x: Domain = [-1,1], Range = [-π/2, π/2]
• arccos x: Domain = [-1,1], Range = [0, π]
• arctan x: Domain = ℝ, Range = (-π/2, π/2), Horizontal asymptotes at y = ±π/2

🎯 Crucial Limits for Calculus

These limits are fundamental to calculus:

lim[x→0] (sin x)/x = 1

lim[x→0] (1 - cos x)/x = 0

lim[x→0] (tan x)/x = 1

lim[x→0] (1 - cos x)/x² = 1/2
Pro Tip: The first limit (sin x)/x → 1 as x → 0 is perhaps the most important limit in all of calculus. It's used to derive the derivatives of sine and cosine functions!

🚀 Derivatives and Integrals Preview

Derivatives of Trig Functions: d/dx[sin x] = cos x
d/dx[cos x] = -sin x
d/dx[tan x] = sec²x
d/dx[sec x] = sec x tan x
d/dx[csc x] = -csc x cot x
d/dx[cot x] = -csc²x
Derivatives of Inverse Trig Functions: d/dx[arcsin x] = 1/√(1-x²)
d/dx[arccos x] = -1/√(1-x²)
d/dx[arctan x] = 1/(1+x²)
d/dx[arcsec x] = 1/(|x|√(x²-1))
d/dx[arccsc x] = -1/(|x|√(x²-1))
d/dx[arccot x] = -1/(1+x²)
Basic Integrals (Antiderivatives): ∫ sin x dx = -cos x + C
∫ cos x dx = sin x + C
∫ sec²x dx = tan x + C
∫ csc²x dx = -cot x + C
∫ sec x tan x dx = sec x + C
∫ csc x cot x dx = -csc x + C
💡 Study Strategy: Practice these concepts regularly! Trigonometry is the foundation for limits, derivatives, integrals, series, and differential equations in calculus.
